Simple scripts to send and read messages from Kafka.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

### Send Messages

1. Edit `send_message.py` - change `TOPIC` and `MESSAGE` variables at the top
2. Run:

```bash
python send_message.py
```

### Read Messages

1. Edit `read_messages.py` - change `TOPIC` variable at the top
2. Run:

```bash
python read_messages.py
```

Press Ctrl+C to stop reading.

That's it!
